miyagawtools
============

miyagawtools is many scripts for bash that has been written by @miyagaw61.

install
-------

<pre>
git clone https://github.com/miyagaw61.git $HOME/miyagawtools/
</pre>

tutorial
--------

* allmv

<pre>
Usage: allmv src...
 src : src file path
</pre>

* allrm

<pre>
Usage: allrm file...
 file : file name that you want to remove
</pre>

* aslr

<pre>
Usage: aslr [option]
 NoUseOptions : check now aslr
 on           : aslr on
 off          : aslr off
</pre>
